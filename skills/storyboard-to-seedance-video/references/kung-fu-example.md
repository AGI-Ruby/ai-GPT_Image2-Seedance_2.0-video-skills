# Kung Fu Example

This file captures the conversation example as a reference case.

## Why this example is useful

It demonstrates a strong version of the two-stage pipeline:

- `image1` = locked female kung fu character
- `image2` = 12-panel storyboard sheet
- storyboard = action order + camera rhythm + escalation map
- video prompt = execution layer over the storyboard

## What to reuse from it

- start directly in action
- every panel contains strong visible momentum
- action and element intensity escalate together
- environment stays atmospheric, not overcrowded
- final frame is locked to a specific airborne climax state

## What not to reuse as a hard default

Do not assume all users want:

- kung fu
- female character
- ancient temple
- fire / dust / stone / water progression

These are example values, not required defaults.
