# X Post Format

Use this format when the user wants promotional posts for the workflow, storyboard, prompt share, or final video.

## Default Three-Post Sequence

This format mirrors the reference pattern from the conversation:

1. **Seedance Prompt Share**
   A post that shares the final Seedance video prompt.
   Media: final video or a representative finished still.
2. **Storyboard Prompt Share**
   A post that shares the GPT Image 2 storyboard prompt.
   Media: the storyboard sheet image.
3. **Maker Note / Learning Post**
   A lighter post that explains what was being tested, what system or framework was used, and what the creator learned.
   Media: final video, best still, or storyboard-to-video pairing.

## Output Structure

When generating X posts, default to this structure:

```text
Post 1:
[seedance prompt share post]

Suggested media:
[what to attach]

Post 2:
[storyboard prompt share post]

Suggested media:
[what to attach]

Post 3:
[maker note / learning post]

Suggested media:
[what to attach]
```

## Tone Rules

- Sound like a real creator sharing process, not a brand account writing launch copy.
- Keep the writing concise and readable.
- Let the prompt itself do the heavy lifting in prompt-share posts.
- The maker-note post should feel reflective and experimental, not overconfident.
- Avoid corporate CTA language.

## Character of Each Post

### Post 1: Seedance Prompt Share

Purpose:

- show the final execution prompt
- frame the video as a controlled result of a prompt system

Recommended pattern:

```text
[Title / label]

[project or character name]

[created with / made on line if useful]

[the Seedance prompt]
```

### Post 2: Storyboard Prompt Share

Purpose:

- show that the video was planned through a storyboard stage
- teach people that storyboard-first prompting improves control

Recommended pattern:

```text
[Title / label]

[one-line setup]

[the GPT Image 2 storyboard prompt]
```

### Post 3: Maker Note / Learning Post

Purpose:

- humanize the process
- share a technique, experiment, or insight
- create a softer entry point for engagement

Recommended pattern:

```text
[Title]

[what was tested]

[what seems to work]

[what still needs more testing]

[brief explanation of the framework or system]
```

## Media Pairing Rules

- If the post shares the Seedance prompt, attach the finished video or output clip.
- If the post shares the storyboard prompt, attach the storyboard sheet image.
- If the post is a maker note, attach whichever media best supports the insight:
  - finished clip if the insight is about motion quality
  - storyboard if the insight is about planning
  - both if the point is translation from planning to execution
