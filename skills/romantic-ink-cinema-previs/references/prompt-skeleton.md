# Prompt Skeleton

Use this when the user wants a single reusable prompt instead of a timed breakdown.

```text
Lens:
[镜头类型]

Camera:
[镜头运动逻辑]

Character Motion:
[角色动作]

Energy Interaction:
[能量如何流动 / 附着 / 扩散 / 堆积 / 回流]

Spatial Emotion:
[空间情绪变化]

Visual Density:
[画面密度变化]

SFX:
[节奏 + 空间 + 动作声音]
```

## Expansion Rules

- If the user asks for romance, default to pale ink light, fingertip glow, soft halos, and gentle drift.
- If the user asks for intensity, increase push-pull, orbit pressure, density stacking, and release force.
- If the user asks for tenderness, reduce motion amplitude and increase breath, stillness, and negative space.

## One-Line Compression Formula

When generating a single paragraph prompt, compress the template into:

```text
[lens + camera participation] + [body rhythm] + [energy physics] + [spatial pressure] + [density arc] + [SFX tone]
```
